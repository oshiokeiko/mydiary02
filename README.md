# 📔 My Diary（初心者向け・スッキリ版）

## どんなアプリ？

ブラウザだけで動く、シンプルな日記アプリです。
書いた日記はブラウザの中（ローカルストレージ）に保存されるので、リロードしても消えません。
サーバーは不要で、`index.html` をダブルクリックするだけで使えます。

**学習課題として使っている技術：**

- JavaScript（配列・オブジェクト・繰り返し・条件分岐）
- jQuery（画面の操作・ボタンの反応）
- ローカルストレージ（保存・削除）
- JSON（`JSON.stringify` / `JSON.parse`）
- Tailwind CSS（見た目）

---

## できること

- **日記を書く**：タイトル・気分の絵文字・本文を入力して追加
- **一覧表示**：新しい順に並ぶ
- **削除**：各カードの「削除」ボタンで1件ずつ消せる
- **保存**：ローカルストレージに自動保存（閉じても消えない）

> シンプルさを優先し、「編集機能」と「ポップアップ画面（モーダル）」はあえて付けていません。
> 入力フォームは常に画面に表示しています。

---

## 📚 はじめての人へ：読む順番

このアプリは**段階的に**作れるようになっています。
いきなり完成版（`app.js`）を読むより、`steps/` フォルダを順番に見るのがおすすめです。

1. `steps/step1.html` … 書く＋一覧（配列・オブジェクト・繰り返し）
2. `steps/step2.html` … ＋ローカル保存（localStorage・JSON）
3. `steps/step3.html` … ＋削除（id・filter）
4. `index.html`（完成版）… ＋気分の絵文字・新しい順・日付

くわしい解説は **[LEARN.md](LEARN.md)** を読んでください。

## ファイル構成

| ファイル | 役割 |
|----------|------|
| `index.html` | 完成版の画面（見た目） |
| `app.js` | 完成版の動き（JavaScript） |
| `style.css` | Tailwindで補えない飾り（全ファイル共通） |

## 使用技術

| 項目 | 技術 |
|------|------|
| マークアップ | HTML5 |
